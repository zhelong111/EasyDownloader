# EasyDownloader
快速下载，支持弹性线程下载，断点续传，丰富的回调支持


Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.zhelong111:EasyDownloader:Tag'
	}
